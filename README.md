# frontend-nanodegree-arcade-game

## Download
* On the right side of the Github page, click "Clone or download"
* Choose "Download Zip" to download this Arcade game to your computer
* Extract the frontend-nanodegree-arcade-game-master.zip file

## How to play
* Launch the game by opening the index.html file
* Use arrow keys on your keyboard to control your character to avoid the bugs
* If your character is hit by a bug, you will be sent back to the starting point.
